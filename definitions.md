# Definitions 

- The CSV (Comma Separated Values) file format allows storing data in tabular form
- A measure is a variable on which operations can be performed.
- An aggregation function is a function that takes multiple values of the same nature as input and returns a single value as output.
- Measures are variables that take numerical values, represent quantities, and have meaning when arithmetic operations are applied to them.
- Dimensions are variables that are not measures. They represent categories.
